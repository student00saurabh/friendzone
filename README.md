# FriendZone 🌐

**FriendZone** is a full-stack social media web application built with Node.js, Express, MongoDB, and Bootstrap. It allows users to connect, post updates, like content, send messages, and build friendships.

---

## 🔗 Live Project

👉 [FriendZone on Render](https://friendzone-7tjx.onrender.com)

👉 [FriendZone GitHub Repository](https://github.com/student00saurabh/friendzone)

---

## 🚀 Features

- ✅ User Registration & Login (with sessions)
- 📝 Post creation, display, like/unlike
- ❤️ Like system with user tracking
- 💬 Real-time styled messaging (sent/received with seen status)
- 🔔 Notification-ready structure
- 📱 Responsive UI using Bootstrap
- 👀 Seen/unseen message handling
- 🧑‍🤝‍🧑 Sender listing with unique conversations
- 🗂 Clean directory structure and modular code

---

## 📁 Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: EJS, Bootstrap 5, HTML/CSS
- **Database**: MongoDB + Mongoose
- **Authentication**: Express-session
- **Deployment**: Render
